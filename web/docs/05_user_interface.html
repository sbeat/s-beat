Benutzeroberfläche

Die S-BEAT Oberfläche baut auf zwei Navigationsebenen auf. Die Hauptnavigation ist horizontal ausgeführt und als roter Balken zu jeder Zeit sichtbar. Damit dem Anwender immer ersichtlich ist, wo er sich derzeit befindet, ist der jeweils aktive Menüpunkt hierbei dunkelgrau hinterlegt. Zu manchen Menüpunkten gibt es eine spezifische Unternavigation, die sich vertikal auf der linken Seite erstreckt. Auch hier wird der jeweils aktive Eintrag hervorgehoben, in diesem Fall in rot. Eingerückte Einträge stellen hierbei Unterthemen des letzten nicht eingerückten vorangegangenen Menüpunktes dar.

Die von S-BEAT erfassten Daten und durchgeführten Analysen wurden in die Bereiche

  • Studiengänge
  • Studenten
  • Prüfungsleistungen
aufgeteilt.

Übersicht

In jedem Bereich startet man mit einer Listenansicht, die man nach unterschiedlichen Kriterien anpassen kann. In allen Bereichen lassen sich Sortierung und die angezeigten Spalten wählen. Die Sortierung kann über einen Klick auf die jeweilige Spalte oder über den Punkt "Einstellungen" vorgenommen werden. Über letzteren lässt sich auch eine Sekundärsortierung sowie die Anzahl der angezeigten Elemente pro Seite einstellen.

In den Bereichen "Studenten" sowie "Prüfungsleistungen" kann man darüber hinaus Filter definieren. Filter sind für alle erfassten Daten im System möglich. Allerdings gibt es Unterschiede in der Handhabung. Für manche Daten wurden Grenzen definiert, nach denen man filtern kann. Andere Daten (z.B. Geschlecht) sind ohnehin auf bestimmte Werte begrenzt, die sich dann dementsprechend auswählen lassen. Zuletzt gibt es noch Daten bei denen man selbst Werte eingeben kann, nach denen gefiltert werden soll.

Aktive Filter werden über der Liste eingeblendet. Sie können mit Klick auf das Merkmal bearbeitet oder über das kleine rote x jederzeit wieder entfernt werden.

Details

Durch Klick auf die ID eines Eintrages kommt man auf die zugehörige Detailseite. Hier finden sich Auflistungen aller erfassten und berechneten Daten für das gewählte Objekt, jeweils sortiert und aufbereitet nach den entsprechenden Themen. Desweiteren gibt es zur besseren Veranschaulichung je nach Bereich unterschiedliche Graphen sowie im Falle der Studenten die Aufschlüsselung der Risikopfade, welche dem Studenten zugeordnet wurden.

Einstellungen

In der Oberfläche vorgenommene Einstellungen, also gewählte Filter, angezeigte Spalten und die jeweilige Sortierung, werden im Browser gespeichert. Erst wenn man seinen Cache leert oder unter Einstellungen auf Zurücksetzen klickt, gehen diese verloren.
