Datenschutz & Zugriffsberechtigungen

Da aus Gründen des Datenschutzes nicht jeder Benutzer Zugriff auf alle Daten haben soll, wird dieser mittels Zugriffsberechtigungen geregelt.
So sind insbesondere Daten, welche Personen identifizieren, wie z.B. Geburtsdatum und Details zur Hochschulzugangsberechtigung nicht in allen Benutzerrollen einsehbar.

Das S-BEAT System ist mit einem Benutzerauthentifizierungssystem gesichert. Jeder Benutzer muss sich mit seinem Benutzeraccount anmelden. Dabei stellt das System fest, ob der Benutzer auf S-BEAT zugreifen darf und welche Berechtigungen er besitzt.

Berechtigungskonzept

Berechtigung Beschreibung
Administration Der Bereich der Administration ist zugänglich und es können dort Datenimporte vorgenommen werden.
Persönliche Daten Persönliche Daten von Studenten können eingesehen und gefiltert werden.
Dazu gehören Geburtsdatum, Geschlecht und Hochschulzugangsberechtigung.
Identifizierende Daten Identifizierende Daten von Studenten können eingesehen werden.
Dazu gehören Matrikel-Nummer, Vorname, Nachname und E-Mail.
Studiengangseinschränkung Ein Benutzer kann nach Studiengangsgruppe eingeschränkt werden.
So sieht dieser nur die Daten der ihm zugewiesenen Studiengangsgruppe.
Diese Einschränkung ist vor allem für Studiendekane vorgesehen.

Übersicht der Benutzerrollen

Jedem Benutzer kann eine Benutzerrolle zugewiesen werden. Welche Berechtigungen die jeweilige Benutzerrolle hat können Sie der Benutzerrechte-Tabelle auf der Benutzer-Seite in der Administration entnehmen.
