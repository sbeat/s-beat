Problembehandlung

Browser Kompatibilität

Die S-BEAT Weboberfläche ist mit HTML5 und CSS3 erstellt. Deshalb kann diese nur mit modernen Browsern genutzt werden. Dazu gehören Internet Explorer ab Version 9, Firefox ab Version 10, Google Chrome, Safari ab Version 4. Getestet und entwickelt wurde S-BEAT mit Firefox 31 und Chrome 37.

Sollte es in modernen Browsern zu Darstellungs- oder Bedienungsproblemen kommen, benachrichtigen Sie bitte das Betreuungsteam von S-BEAT.

Datenupdate nimmt CSV Datei nicht an

Wurde eine Datei ausgewählt und es erscheint ein Fehler, wurde diese abgelehnt.
Jede Datei wird nachdem sie ausgewählt wurde überprüft. Je nach Art werden verschiedene Merkmale geprüft.

Folgendes wird geprüft:

  1. Der Dateiname einer Datei darf nur Buchstaben, Zahlen, - und _ enthalten und muss mit .csv enden.
  2. Die Datei muss mindestens eine Datenzeile enthalten.
  3. Sind alle geforderten Spalten vorhanden und haben diese die richtige Groß/Kleinschreibung?
  4. Sind Datumsangaben vorhanden und haben diese das geforderte Format?

Folgende Fehlermeldungen können vorkommen:

Fehler Lösung
Der Dateiname darf nur Buchstaben, Zahlen, - und _ enthalten und muss mit .csv enden. Benennen Sie die Datei um und entfernen Sie alle nicht erlaubten Zeichen.
Keine Datenzeilen in der Datei gefunden. Die ausgewählte Datei hat offensichtlich keine Daten. Wählen Sie eine andere Datei oder überprüfen Sie, ob es bei der Erstellung der Datei ein Problem gab.
Das Trennzeichen ";" wurde nicht gefunden. Die Datei sollte im CSV Format sein. Dabei werden Spalten mit einem Trennzeichen getrennt. Doch das gesuchte Trennzeichen wurde nicht gefunden. Die Spalten wurden möglicherweise mit einem anderen Trennzeichen getrennt. Prüfen Sie das Format der CSV Datei.
Die Spalte "NAME" wurde nicht gefunden. Es wird eine bestimmte Spalte erwartet. Diese Spalte fehlt offensichtlich in der CSV Datei. Bitte fügen Sie diese Spalte in der Datei ein oder prüfen Sie woher die Datei kommt.
Die Zeile 'x' hat weniger Spalten als es Kopfspalten gibt. Das CSV Format scheint nicht konsistent zu sein. Möglicherweise ist die CSV Datei beschädigt. Möglicherweise gab es ein Problem mit der Erstellung der Datei.
Die Spalte "NAME" enthält ein ungültiges Datumsformat. Das Datum sollte im Format TT-MMM-YY sein Z.B. 04-Mrz-99. Das Datum in der Datei ist nicht wie erwartet formatiert. Passen Sie die Erstellung der Datei entsprechend an.
Die Spalte "NAME" enthält ein ungültigen Monat "x". Das Datum in der Datei ist nicht wie erwartet formatiert. Passen Sie die Erstellung der Datei entsprechend an.

Datenimport läuft nicht durch

Sollte der Datenimport nicht durchlaufen, kann das viele Ursachen haben. Nachfolgend werden ein paar Anhaltspunkte erläutert.
In den meisten Fällen, muss dies von einem Entwickler untersucht werden. Als Nutzer können Sie nur das Format der hochgeladenen CSV Dateien überprüfen.

Schritt Beschreibung
import_courses Läuft dieser Schritt nicht durch, könnte es daran liegen, dass die Datei data/courses_2014.csv fehlt oder diese einen unerwarteten Zeichensatz besitzt.
Weil dies der erste Schritt ist, könnte es auch daran liegen, dass die Datenbank nicht erreichbar ist oder die Festplatte voll ist und die Datenbank keine weiteren Einträge annimmt.
import_students Eine der beim Datenimport unter Studentendaten oder Studenten Identifikationsdaten hochgeladenen Datei könnte einen unerwarteten Zeichensatz haben. Sie könnte auch Daten enthalten, die so nicht erwartet wurden.
import_exams Eine der beim Datenimport unter Prüfungsleistungsdaten hochgeladenen Dateien könnten unerwartete Daten haben.
calculate_student_exams Vermutlich ist eine Rechnung fehlgeschlagen. Eine ungewöhnliche Konstellation aus Studenten und Prüfungsleistungsdaten. Dies ist allerdings sehr unwahrscheinlich.
generate_definitions Sehr unwahrscheinlich, dass hier ein Fehler auftritt.
generate_paths_apriori Dieser Schritt läuft in mehrerern Prozessen ab, welche die Systemhardware stark belasten. Wenn es kein Problem des Programms ist, könnte es ein Hardwareproblem sein.
calculate_student_risk Vermutlich ist eine der komplexen Berechnungen fehlgeschalgen auf Grund einer unvorhergesehenen Konstellation.
